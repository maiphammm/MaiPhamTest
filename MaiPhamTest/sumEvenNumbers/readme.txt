Sum of Even Numbers
This project is a simple JavaScript program that calculates the sum of all even numbers from 1 to a given positive integer n. It provides a user interface to enter the integer and displays the result.

Features
Input Handling: Allows the user to input a positive integer.
Calculation: Computes the sum of all even numbers from 1 to the input integer.
Display: Shows the result on the webpage.
Files
index.html: The main HTML file that provides the structure and user interface.
sumEvenNumbers.css: The CSS file that styles the HTML elements.
sumEvenNumbers.js: The JavaScript file that contains the logic for calculating the sum and updating the display.
Setup
Download the Files: Ensure you have the following files in the same directory:

index.html
sumEvenNumbers.css
sumEvenNumbers.js
Open the HTML File: Open index.html in a web browser. You should see the user interface for entering an integer and calculating the sum.

Usage
Enter a positive integer in the input field.
Click the "Calculate Sum" button.
The result, which is the sum of all even numbers from 1 to the entered integer, will be displayed below the button.
